# Campus Companion

Campus Companion is a modern and accessible web application for first-year students. It provides timetable viewing, events, society events, notifications, a campus map, lost and found, helpdesk support, and accessibility settings.

## Chosen stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Supabase
- Netlify for deployment

## How to run locally
Clone the repository
- git clone https://github.com/A00029255/Enterprise_CA3.git
- cd Enterprise_CA3
- npm run dev
